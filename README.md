# KoraVault Final: Cardano Savings DApp

Full-stack DApp with Haskell (Plutus) logic and Lucid integration.

## Files
- `index.html`: Main UI (Root)
- `app.js`: Logic (Root)
- `contracts/KoraVault.hs`: Haskell Script
- `vercel.json`: Vercel configuration

## Deployment
1. Upload to GitHub.
2. Link to Vercel.
3. Vercel will serve the root directory automatically.
