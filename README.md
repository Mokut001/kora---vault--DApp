# KoraVault: Professional Haskell Savings DApp

A full-stack savings application for the Cardano blockchain. It allows users to set goals and lock ADA behind a smart contract.

## Project Structure
- **/contracts**: Contains `KoraVault.hs` - the Haskell logic that enforces the "Save till Target" rule.
- **/public**: The Vercel-ready frontend.
- `app.js`: High-performance Lucid logic.

## Deployment to Vercel
1. Upload this project to a GitHub Repository.
2. Link it to Vercel.
3. Your DApp is live.

## Tech Stack
- Smart Contract: Haskell (Plutus V2)
- Frontend: HTML5, Tailwind CSS
- Blockchain SDK: Lucid-Cardano
